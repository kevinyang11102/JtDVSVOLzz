# 音乐后台管理系统 java984

## 项目概述

音乐后台管理系统是基于B/S架构的MVC模式设计，采用Springboot和Vue技术栈开发，旨在为个人学习使用提供音乐管理平台。

## 技术栈

### 服务端

- Springboot
- Mybatis-Plus

### 前端

- Vue
- axios
- Element-UI

### 推荐运行环境

- JDK 1.8
- Maven
- MySQL
- Node.js

## 目录结构

```
项目根目录
├── backend                后端代码目录
│   ├── src                源代码目录
```
